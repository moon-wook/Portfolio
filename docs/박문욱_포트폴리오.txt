Project      03. HI WIPLAT


Introduce project

     작업  기간        2025. 06. 10 ~ 현재(진행중)

인력    구성(기여도)      AI 1명  (AI 기여도  100%)

                   웹사용자가마우스나키보드입력없이음성명령만으로자사웹페
                   이지를   탐색하고    제어할    수있도록, 음성     인식(STT)과   대규모   언어
   프로젝트목적
                   모델(LLM)을   결합한   AI Agent를개발하여     웹  접근성과    사용자   경험
                   을  향상시키는     것.

                   •트리거     알고리즘을통해       사용자   맞춤형작동      구현
   프로젝트    내용      •OpenAI GPT 모델과     Playwright 브라우저  자동화    기술을   연동
                   •사용자의자연어         명령을해석하고       실제웹페이지       조작으로연결

                   1)트리거     알고리즘    구현  – 음성  또는   텍스트   입력  내  특정  키워드
                      를실시간감지하여서비스동작을자동으로시작하도록설계
                   2)LLM 연동    및명령어     파싱  – 프롬프트    엔지니어링을      통해  사용
     주요  업무           자  명령을   해석하고, 대응되는       페이지   URL 및  실행  동작을   동적
   및상세역할              으로생성
                   3)웹페이지     자동   제어  기능   구현  – Playwright를활용해   자사   웹페
                      이지의    메뉴  이동, 데이터    정렬, 오디오    재생   등사용자의     명령
                      에맞는기능을자동수행

     사용언어
                   Python, OpenAI, Playwright, STT(Whisper), TTS
   및개발환경

                   기술적성과
                    - LLM + 브라우저    자동화결합      성공
                    - 실시간   웹  조작  AI 에이전트    구현
                    - 모듈형   아키텍처    설계   → 타  웹환경에도확장       가능
  프로젝트의의및
                    - 자연어   명령  기반   인터페이스     실현성   검증
     향후계획
                   향후계획
                    - 다국어   음성  명령   지원  확대
                    - 사용자   맞춤형   명령   학습  기능   추가
                    - 복잡한   멀티스텝    자동화    및웹   내장형음성     인터페이스     구현

인력    구성(기여도)        BE 2명/ FE 1명  / AI 5명  (AI 기여도   40%)


                     AI 모델을   통한   문제  해결   역량을    키우기    위해  직접   데이터를     수집
   프로젝트목적
                     하여사용자의편의성을보장하기위해효과적인해결책을모색함

                                                                                                                                                            최종모델성능
                     •사용자     자녀의연령대에최적화            된  놀이정보     제공
                     •사용자     거주지역     별  육아   정책   정보제공
                     •사용자에게방문          지역  및  프롬프트를입력받아
   프로젝트내용
                     •AI를   활용하여    아이와    함께할    수  있는  장소   추천
                     •사용자     거주지역     긴급   의료   기관   정보제공
                     •사용자     거주지역     주말   어린이집정보        제공

                     1)네이버     플레이스와     카카오    맵에서    리뷰   데이터를     Selenium을
                        활용해크롤링하여수집
                     2)한국어     기반의   ko-BERT 모델을    사용해    리뷰를    긍정과    부정으로
                        분류하려     했으나, 만족스러운        성능을   얻지   못해서    이를   해결하
     주요   업무            기위해    기존  ko-BERT모델을     파인튜닝하여       리뷰를   재학습시켰고,
    및상세역할               성능을크게개선함
                                                                                          기존  ko-BERT모델                     fine-tunning 후모델
                     3)각   장소에   대해   ai평점   추출
                     4)긍정    리뷰  데이터를     자체   토큰화    하여  중요키워드      추출
                     5)중요키워드로        이루어진     워드클라우드      구현
                     6)도로명주소를        이용해    좌표로   변환(kakao api 사용)

      사용언어
                     Python, Pandas, Numpy, Pytorch, Selenium, Colab
